# Smart Project Tagging Demo

This example demonstrates how a simple hackathon project (an idea tracker) can be transformed into an AI-powered workspace. It features a basic rule-based system that automatically analyzes project idea descriptions and suggests relevant tags or categories, simulating an AI's ability to add smart features. This showcases the concept of enhancing existing applications with intelligent automation.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open a terminal or command prompt.
3. Navigate to the directory where you saved the file.
4. Run the script using `python main.py` and follow the on-screen prompts.
